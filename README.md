# quant-trading

量化交易课程作业与练习代码仓库。

## 目录

- `TASK1/`：量化交易基础概念、Tushare 数据获取、收盘价曲线图和 CSV 数据。
- `TASK2/`：数据诊断、RSI、MACD、布林带、KDJ 指标计算、图表和指标 CSV。
- `TASK3/`：双均线策略（金叉/死叉信号）、回测与 MDD/夏普/累计回报评估、多股票多参数对比。
- `TASK4/`：海龟交易策略：唐奇安通道、ATR、买卖信号与止损、回测（累计收益/最大回撤/夏普）及通道周期、止损倍数、股票类型参数敏感性分析。
- `docs/`：GitHub Pages 作品集展示页，使用公开图表和指标 CSV 做网页展示。
- `private_submissions/`：本地提交 PDF，已被 `.gitignore` 忽略，不上传 GitHub。

后续作业按 `TASK5/` 等目录继续追加。
